# Reminder service which sends you notifications
## To run server just start docker containers with `docker-compose up` in the root folder.
## .env file should be in the rood folder
#### fields of env file:
* APP_SECRET_KEY
* DJANGO_DEBUG
* DB_ENGINE
* DB_NAME
* DB_USER
* DB_PASSWORD
* DB_HOST
* DB_PORT
* EMAIL_HOST_USER
* EMAIL_HOST_PASSWORD
